Requirements:

Cmake 4.2.1 or newer

C/C++ Compiler:g++, clang or msvs



How to Run



Clone this repository:

git clone https://github.com/EinstYou/SDL_Terraria_Clone.git





Clone librayies in project folder:

git clone https://github.com/libsdl-org/SDL.git vendored/SDL

git clone https://github.com/libsdl-org/SDL_image.git vendored/SDL_image






Then build and run:

cmake -B build

cmake --build build

build\Terraria.exe
